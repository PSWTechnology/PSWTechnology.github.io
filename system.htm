PSW Sound Monitoring System

This system was featured in the Sound and Vibration article "Cost-Effective Wireless System for Large-Scale Noise Monitoring".

We presented a paper on "Automatic Wireless Sound Monitoring System and Its Application at Caldecott Tunnel" at Noise-Con 2011.

One of the challenges of remote sound monitoring is real time data access. PSW-Tech has solved the challenge by using a 3G sound monitoring station that uploads real time sound data to an internet web site. The system is installed at 10 monitoring stations at Caldecott Tunnel in California.

Real Time Sound Data Website

Live data is transmitted wirelessly to a website where sound levels can be reviewed at any time. Time averaged sound (Leq) and maximum sound (Lmax) levels are graphed for easy review as well as a searchable history of sound data.

Visit a demonstration at our client’s site. www.mei-wu.com/pswdemo

Instant Alarm and Audio Recordings

The sound monitoring system provides instantaneous notification via text message (SMS) and/or e-mail to any mobile phone or computer when the sound level criteria are exceeded. It also provides audio recordings of exceedance events to identify the source of the event.

Wireless Data Transmission (3G/Wi-Fi)

Sound data can be transmitted to the real time website via a 3G data connection or a Wi-Fi computer network. This allows the monitoring system to be completely wireless in remote locations where 3G/Wi-Fi signals are available.

Fully Customizable Set-ups

Each sound monitoring system can independently send data to the website which allows for up to 30 sound monitors to be operating simultaneously. The sound monitor can also be adapted to be a vibration monitor for excavation/blasting projects.
